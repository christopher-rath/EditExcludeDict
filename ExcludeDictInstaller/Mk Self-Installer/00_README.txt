After successfully building a Release version of the installer, there
will be two files in the the "ExcludeDictInstaller/Release/" folder:
 * ExcludeDictInstaller.msi
 * setup.exe

Open a Windows CMD window, cd into this directory, and then run the
"mkInstall.cmd" script.

This will create "ExcludeDictSetup.exe".  That .exe file may then be
used to install the MS Word ExcludeDictionary Addin.
