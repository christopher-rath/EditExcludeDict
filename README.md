# Edit Exclude Dict

An unadvertised (by Microsoft) feature of MS Word is the ability to
configure a list of words—on a per-language basis—that are always
flagged as misspelled.  The problem with this feature is that it is
almost impossible for regular users to access: it involves digging
into hidden folders and editing obscurely-named files.

*This add-in enables editing of those files by regular users.*

Download and run the **ExcludeDictionarySetup.exe** file to install
the Addin.  It will add a new "Exclude Dict" group to Word's Review
tab on the ribbon.
